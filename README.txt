== Description ==

News Letter that collates the News, it uses custom post type to create news and newsletter.
It is dependent on ACF and Event Organiser plugins.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `tm-newsletter.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
